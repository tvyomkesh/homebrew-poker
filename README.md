# Homebrew Poker

Homebrew tap for [Poker](https://github.com/tvyomkesh/poker) - a terminal-based Texas Hold'em game.

```
    ♠ ♥ ♦ ♣
   POKER
```

## Installation

```bash
brew tap tvyomkesh/poker
brew install poker
```

Or in one command:

```bash
brew install tvyomkesh/poker/poker
```

## Usage

```bash
poker
```

## What You Get

- **Texas Hold'em** in your terminal
- **1-3 AI opponents** with Easy/Medium/Hard difficulty
- **Real-time odds** and hand rankings
- **Beautiful Unicode cards** with colors

## Requirements

- macOS with Homebrew
- Python 3.12 (installed automatically by Homebrew)
- Terminal size: 140 x 40 minimum

## Updating

```bash
brew update
brew upgrade poker
```

## Uninstalling

```bash
brew uninstall poker
brew untap tvyomkesh/poker
```
